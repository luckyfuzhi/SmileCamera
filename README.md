# SmileCamera

## 一、demo概述

本demo是一款结合华为深度学习SDK的简单多功能相机，可以选择调用系统相机并采用系统自带的滤镜等相机功能进行拍照，也可以通过调用笑脸识别抓拍相机，在被拍摄的人露出笑容的时候进行抓拍。同时还有内置相册功能进行照片展示。
